# netutils-docker

A small container image to test Networking.

Supported arch:
- linux/amd64

Included packages:
- bind-tools
- curl
- dog
- gping
- htop
- ipcalc
- iputils
- mtr
- netcat-openbsd
- nmap
- sipcalc
- screen
- tcpdump
- traceroute
- openssl
- whois
- wget

How to get image:

```sh
docker pull ghcr.io/deanvry/netutils:main
```
